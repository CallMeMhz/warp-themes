Warp Themes

Beautiful color themes for Warp Terminal

Installation

  1. Clone or download the repository from GitHub
  2. Copy theme files to your Warp themes directory:
cp themes/*.yaml ~/.warp/themes/
  1. Open Warp Terminal
  2. Go to Settings → Appearance → Theme
  3. Select your desired theme from the list
